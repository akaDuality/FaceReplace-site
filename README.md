# Face Replace — landing site

Static landing page for the **Face Replace · No Limits** iOS / macOS app.

- Single file: `index.html` (vendor-free, only loads Google Fonts)
- App icon: `app-icon.png`
- No build step

## Local preview

```sh
open index.html
```

## Hosting

Served via **GitHub Pages** from the `main` branch root.

Live URL: <https://akaduality.github.io/FaceReplace-site/>

`.nojekyll` is present so GitHub Pages serves files as-is (no Jekyll processing).
